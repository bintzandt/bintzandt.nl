This is the repo for my own website, hosted at [bintzandt.nl](https://bintzandt.nl).

This site is build using the Lumen Gatsby starter, which can be found at https://github.com/alxshelepenok/gatsby-starter-lumen.

# Installation
You can start from this repository using  
```gatsby new my-website https://github.com/bintzandt/bintzandt.nl```
